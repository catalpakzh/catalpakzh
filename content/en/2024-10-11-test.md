## Make the footnote appear on the right

This function let the footnote show on the right side of the text. You can find some useful information in blog of yihui[^1]. 

[^1]: This is his github [question section](https://github.com/yihui/yihui.org/discussions/1561).

At first you should insert the code below into any `html` file.

```
<script src="https://cdn.jsdelivr.net/npm/@xiee/utils/js/sidenotes.min.js" defer></script>
```
